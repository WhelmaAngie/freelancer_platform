# Freelancer Platform (Django)

A Django web application that demonstrates a simple freelancer/job listing platform with basic CRUD operations and template-based views.

## Features
- Create and list freelancers
- Create and list jobs
- Simple UI with static assets
- Django admin support

## Tech Stack
- Python
- Django
- SQLite (development)
- HTML/CSS
